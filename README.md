# Troop 399 Scoutbook Token

A lightweight Chrome extension for Scout leaders that retrieves, validates, and copies the current Scoutbook session token.

## Features

- Reads the current Scoutbook session token
- Decodes the JWT locally
- Displays the signed-in user
- Displays session expiration
- Shows expiration time
- Copies the token to the clipboard
- No token is stored or transmitted

## Permissions

- activeTab
- scripting
- clipboardWrite

The extension only operates on official Scouting domains.

## Privacy

This extension does not collect, transmit, or store any personal data or authentication tokens.
