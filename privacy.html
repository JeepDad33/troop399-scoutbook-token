Privacy Policy

Troop 399 Scoutbook Token

Last Updated: June 30, 2026

Overview

Troop 399 Scoutbook Token is a lightweight Chrome extension that retrieves the authenticated Scoutbook session token from the currently open Scoutbook page, displays session information, and copies the token to the clipboard for use with authorized Troop 399 tools.

Information We Collect

This extension does not collect, transmit, sell, share, or store personal information.

No analytics, advertising, telemetry, tracking, or remote logging are performed.

Authentication Token

The extension temporarily accesses the existing Scoutbook authentication token stored by the Scoutbook web application.

The token is used only to:

  • Display the signed-in username.
  • Display the remaining session time.
  • Display the session expiration time.
  • Copy the token to the user's clipboard.

The token is:

  • Never transmitted over the Internet.
  • Never stored by this extension.
  • Never logged.
  • Never shared with any third party.
  • Processed entirely within the user's browser.

Permissions

  • activeTab — Accesses the active Scoutbook tab after the user opens the extension.
  • scripting — Reads the existing Scoutbook authentication token already stored by the website.
  • clipboardWrite — Copies the authentication token to the user's clipboard.

Data Sharing

No information is sold, transferred, disclosed, or shared with any third party.

Security

All processing occurs locally inside the user's web browser. The extension does not communicate with external servers.
