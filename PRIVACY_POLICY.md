# Privacy Policy
**Troop 399 Scoutbook Token**

**Last Updated:** June 29, 2026

## Overview

Troop 399 Scoutbook Token is a Chrome extension that retrieves the authenticated Scoutbook session token from the currently open Scoutbook page, displays basic session information, and copies the token to the user's clipboard.

## Information We Collect

This extension does **not** collect, store, transmit, or share any personal information.

No analytics, tracking, telemetry, or advertising technologies are used.

## Authentication Token

The extension temporarily accesses the Scoutbook session token that already exists in the user's browser session for the sole purpose of:

- Displaying the signed-in username
- Displaying the session expiration time
- Copying the token to the system clipboard at the user's request

The token is:

- Never transmitted to any external server
- Never stored on disk
- Never logged
- Never shared with any third party

The token exists only in browser memory while the extension is running.

## Permissions

This extension requests the following Chrome permissions:

### activeTab

Used to access the currently active Scoutbook tab.

### scripting

Used to execute a script within the active Scoutbook page to retrieve the existing session token.

### clipboardWrite

Used to copy the session token to the user's clipboard.

## Data Sharing

No information is sold, shared, transferred, or disclosed to any third party.
